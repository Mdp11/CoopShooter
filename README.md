# CoopShooter
Co-op shooter game made in UE4

# Features

- Character movement
  - Walk/Run
  - Sprint
  - Jump
  - Crouch
  - Iron sight
- Weapon system
  - Different weapon types
    - Hitscan
    - Projectile-based
  - Spread
  - Recoil
  - Ammo and reloading
- Animations
